Relative Complexity Adjusment Factor (RCAF)


Relative Complexity Adjusment Factor (RCAF) digunaka untuk menghitung bobot kompleksitas software berdasarkan 14 karakteristik


Penilaian Bobot Kompleksitas memiliki skala 0-5

- 0 = Tidak Pengaruh

- 1 = Insidental

- 2 = Moderat

- 3 = Rata-rata

- 4 = Signifikan

- 5 = Esential

Function Point (FP)

Rumus menghitung Function Point = CFP x (0.6 + 0.01 + RCAF)
